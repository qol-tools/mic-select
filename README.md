# Select Microphone

Switch microphone input sources on Linux (Ulauncher) and macOS (Raycast).

## Installation

### Linux
```bash
make install
```

### macOS
```bash
brew install switchaudio-osx
pip install -r requirements.txt
cd macos/raycast && npm install && npm run build
```

## Usage

**Linux**: Open Ulauncher (`Ctrl+Space`), type `mic `, select microphone.

**macOS**: Open Raycast (`Cmd+Space`), search "List Microphones" or "Switch Microphone".
