# Mic Switcher

Quick microphone switcher.

Works for:
- **macOS**: Raycast (free tier compatible)
- **Linux**: Ulauncher

## Install

```bash
make install
```

**macOS**: After install, run:
```bash
cd macos/raycast && npm run dev
```

## Use

Open launcher, type `mic`, select microphone.

## Uninstall

```bash
make uninstall
```
