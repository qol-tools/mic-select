# Mic Switcher

Quick microphone switcher.

Works for:
- **macOS**: Raycast script commands (free tier)
- **Linux**: Ulauncher

## Install

```bash
make install
```

**macOS**: Add script directory to Raycast (path shown after install).

## Use

Open Raycast, type `List Microphones` or `Switch Microphone`.

## Uninstall

```bash
make uninstall
```
